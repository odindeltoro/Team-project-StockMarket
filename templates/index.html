Objective

Quantopian

We chose this platform for its incredible libraries and data base regarding financial data and we use them to confirm the correlation between QQQ (ETF) and the NASDAQ index. In the end we created an algorithm which could mimic their behavior.

Machine Learning

Create an algorithm using 3 models of machine learning (linear,quadratic regression and KNN) which can predict the returns of an investment and the price of a stock given a specific window of time.
